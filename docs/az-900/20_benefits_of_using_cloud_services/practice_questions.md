# Practice Questions – Cloud Benefits

## 🧠 Question 1

What is high availability?

A) Increasing performance
B) Ensuring services remain accessible
C) Reducing costs
D) Encrypting data

**Answer:** B

---

## 🧠 Question 2

What does scalability allow you to do?

A) Encrypt data
B) Add or remove resources
C) Monitor performance
D) Manage identities

**Answer:** B

---

## 🧠 Question 3

What is elasticity?

A) Manual scaling
B) Automatic scaling based on demand
C) Data encryption
D) Backup process

**Answer:** B

---

## 🧠 Question 4

Which concept ensures a system can recover from failures?

A) Scalability
B) Reliability
C) Elasticity
D) Governance

**Answer:** B

---

## 🧠 Question 5

Which tool helps estimate costs before deployment?

A) Azure Monitor

## 🧠 Question 6

## 🧠 Question 10

What is the difference between scalability and elasticity?

A) No difference
B) Scalability is automatic, elasticity is manual
C) Scalability is manual, elasticity can be automatic
D) Elasticity is only for storage

**Answer:** C

Understand:

* Scalability vs Elasticity
