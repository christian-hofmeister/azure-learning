### 1

Which Azure feature ensures that **only specific VM sizes** can be deployed?
A) RBAC
B) Policy
C) Locks
D) Tags

<details><summary>Answer</summary>

**B) Azure Policy**
👉 Enforces rules (allowed SKUs)

</details>
