# AZ-900 Cheat Sheet (Last-Day Review)

## ☁️ Cloud Concepts

### Benefits of Cloud

- High Availability
- Scalability
- Elasticity
- Reliability
- Predictability
- Security
- Governance
- Manageability

---

### CapEx vs OpEx

- CapEx → upfront cost (on-prem)
- OpEx → pay-as-you-go (cloud)

---

## 🏗️ Cloud Models

### IaaS

- You manage OS & apps
- Example: Virtual Machines

### PaaS

- Azure manages OS
- You deploy apps
- Example: App Service, Azure SQL

### SaaS

- Fully managed
- Example: Microsoft 365

---

## 🧱 Core Azure Services

### Compute

- Virtual Machines → IaaS
- App Service → PaaS (Web Apps)
- Azure Functions → Serverless
- Container Instances → Containers without orchestration

---

### Storage

- Blob Storage → files, images
- Disk Storage → VM disks
- File Storage → shared drives

---

### Networking

- VNet → private network
- Subnet → segmentation
- NSG → traffic filtering
- VPN Gateway → on-prem connection

---

### Databases

- Azure SQL → PaaS relational DB
- Cosmos DB → NoSQL

---

## ⚙️ Scaling Concepts

### Scalability

- Scale UP → more power (CPU/RAM)
- Scale OUT → more instances

### Elasticity

- Auto scale based on demand

---

## 🔐 Security

### Shared Responsibility Model

- Azure → physical infrastructure
- You → data, access, configuration

---

### Identity

- Microsoft Entra ID (Azure AD)
- Authentication & Authorization

---

### Access Control

- RBAC (Role-Based Access Control)

---

## 📊 Monitoring

- Azure Monitor → metrics & logs
- Application Insights → app-level monitoring
- Log Analytics → queries

---

## 💰 Pricing

- Pay-as-you-go
- Reserved instances (cheaper)
- Spot pricing (cheapest, but interruptible)

---

## 🧾 Governance

- Azure Policy → enforce rules
- Resource Locks → prevent deletion
- Tags → cost tracking

---

## 🌍 High Availability

- Regions
- Availability Zones
- Region Pairs

---

## 🔁 Disaster Recovery

- Backup
- Replication
- Geo-redundancy

---

## ⚡ Serverless

- Azure Functions
- Runs on demand
- Pay-per-execution

---

## 🧠 MUST-KNOW (Exam Favorites)

- IaaS vs PaaS vs SaaS
- CapEx vs OpEx
- Scaling vs Elasticity
- High Availability vs Fault Tolerance
- VNet / NSG basics
