# AZ-900 Lab 05 – Create Blob Storage

## 🎯 Objective
Store unstructured data in Azure Blob Storage.

---

## 🧠 Key Concepts

### Blob Storage
- Object storage for files
- Highly scalable

### Storage Account
- Container for storage services

---

## 💡 Exam-Relevant Points

- Blob = unstructured data
- Access tiers:
  - Hot
  - Cool
  - Archive

---

## 🔍 Deep Dive

### Use cases
- Images, videos
- backups
- logs

### Advantages
- cost-efficient
- durable
- scalable

### POIneer relevance
- Map tiles
- SQLite downloads
- data distribution
