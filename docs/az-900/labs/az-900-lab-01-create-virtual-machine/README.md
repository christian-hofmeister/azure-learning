# AZ-900 Lab 01 – Create a Virtual Machine

## 🎯 Objective
Provision and access an Azure Virtual Machine (IaaS).

---

## 🧠 Key Concepts

### Virtual Machine (VM)
- Infrastructure as a Service (IaaS)
- Full control over OS and software

---

## 💡 Exam-Relevant Points

- VM = IaaS
- You are responsible for:
  - OS updates
  - security
  - scaling

---

## 🔍 Deep Dive

### When to use VMs?
- Legacy applications
- Lift-and-shift scenarios
- Full control required

### Downsides
- Maintenance overhead
- Always running → costs

### Comparison
- VM → always running
- Functions → event-driven

### POIneer relevance
- Possible for batch jobs
- Likely overkill
