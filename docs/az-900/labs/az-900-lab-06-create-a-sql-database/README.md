# AZ-900 Lab 06 – Create a SQL Database

## 🎯 Objective
Create and use an Azure SQL Database.

---

## 🧠 Key Concepts

### Azure SQL Database
- Platform as a Service (PaaS)
- Managed relational database

---

## 💡 Exam-Relevant Points

- Automatic backups
- High availability
- Scaling built-in

---

## 🔍 Deep Dive

### Advantages
- No OS management
- Automatic patching
- Built-in security

### Comparison
- VM SQL → full control
- Azure SQL → less maintenance

### POIneer relevance
- Backend APIs
- not ideal for offline usage
