# AZ-900 Lab 02 – Create a Web App

## 🎯 Objective
Deploy a web application using Azure App Service.

---

## 🧠 Key Concepts

### Azure App Service
- Platform as a Service (PaaS)
- Hosting for web apps and APIs

---

## 💡 Exam-Relevant Points

- App Service = PaaS
- No OS management
- Built-in scaling

---

## 🔍 Deep Dive

### Advantages
- Fast deployment
- No infrastructure management
- Integrated monitoring

### Comparison
- VM → more control, more effort
- App Service → less effort

### POIneer relevance
- Ideal for:
  - backend API
  - admin UI
