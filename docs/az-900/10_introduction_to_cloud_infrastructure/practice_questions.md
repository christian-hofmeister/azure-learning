# Practice Questions – Introduction to Cloud Infrastructure

## 🧠 Question 1

Which cloud model provides resources over the internet to multiple customers?

A) Private Cloud
B) Public Cloud
C) Hybrid Cloud
D) Dedicated Cloud

**Answer:** B

---

## 🧠 Question 2

What type of expense is cloud computing typically considered?

A) CapEx
B) OpEx
C) Fixed Cost
D) Capital Lease

**Answer:** B

---

## 🧠 Question 3

Which Azure service provides full control over the operating system?

A) Azure Functions
B) Azure App Service
C) Virtual Machines
D) Containers

**Answer:** C

---

## 🧠 Question 4

Which service is used to connect on-premises networks securely over the internet?

A) ExpressRoute
B) VPN Gateway
C) Azure DNS
D) NSG

**Answer:** B

---

## 🧠 Question 5

Which storage type is best for unstructured data like images?
