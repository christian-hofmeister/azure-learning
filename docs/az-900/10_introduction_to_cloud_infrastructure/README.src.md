# Introduction to Cloud Infrastructure (AZ-900)

This section covers the fundamental concepts of cloud computing and the core Azure architecture and services.

## 📚 Topics Covered

### Cloud Concepts

* What is cloud computing
* Shared responsibility model
* Cloud deployment models (Public, Private, Hybrid)
* Consumption-based pricing model
* Cloud pricing models

### Azure Architecture & Services

* Core architectural components
* Compute services (VMs, containers, functions)
* Networking (VNet, VPN, ExpressRoute, DNS)
* Storage services and redundancy options

### Management and Governance

* Cost management and pricing tools
* Governance (Azure Policy, Resource Locks)
* Monitoring (Azure Monitor, Advisor, Service Health)

### Tools and Deployment

* Azure Portal, CLI, PowerShell
* ARM templates
* Azure Arc

---

## 🎯 Learning Goals

After completing this section, you should be able to:

* Understand core cloud concepts and benefits
* Describe Azure’s global infrastructure
* Identify key Azure services and their use cases
* Explain basic networking and storage concepts
* Understand governance, monitoring, and cost management tools
