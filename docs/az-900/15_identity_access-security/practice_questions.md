# 🧪 AZ-900 Practice Questions (with Traps)

## ❓ Question 1
Which model assumes breach and requires verification for every request?

A) Defense-in-Depth  
B) RBAC  
C) Zero Trust  
D) Conditional Access  

👉 **Answer: C (Zero Trust)**

---

## ❓ Question 2
Which Azure feature controls *who can do what on which resource*?

A) Conditional Access  
B) RBAC  
C) Defender for Cloud  
D) Key Vault  

👉 **Answer: B (RBAC)**

---

## ❓ Question 3
A company wants to allow external partners to access internal resources using their own accounts. Which solution?

A) B2C  
B) B2B Collaboration  
C) RBAC  
D) Conditional Access  

👉 **Answer: B (B2B Collaboration)**

---

## ❓ Question 4 (Trap!)
Which solution is used for customer-facing applications with social logins?

A) B2B  
B) B2C  
C) Direct Connect  
D) RBAC  

👉 **Answer: B (B2C)**

---

## ❓ Question 5
What does Conditional Access evaluate?

A) Only user identity  
B) Only location  
C) Multiple signals (user, device, location, risk)  
D) Only password strength  

👉 **Answer: C**

---

## ❓ Question 6
Which method provides the **highest security and convenience**?

A) Password  
B) SSO  
C) MFA  
D) Passwordless  

👉 **Answer: D (Passwordless)**

---

## ❓ Question 7 (Trap!)

Which service is used to securely store secrets and keys?

Which concept uses multiple layers of security?

## ❓ Question 10 (Trap!)

## 🧠 Tip
👉 Watch for keywords:
- "who can do what" → RBAC  
- "if condition then action" → Conditional Access

- "customers" → B2C  
- "layers" → Defense-in-Depth  
- "verify every request" → Zero Trust
