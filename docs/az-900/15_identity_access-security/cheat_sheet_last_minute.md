# AZ-900 Cheat Sheet – Identity & Security (Last Minute)

## 🔐 Identity Basics

* Authentication = verify identity
* Authorization = grant access
* Microsoft Entra ID = identity provider

---

## 👤 Authentication Methods

* Password
* MFA (Multi-Factor Authentication) ✅ important
* Passwordless (e.g., biometrics)

---

## 🔑 Authorization

* RBAC = Role-Based Access Control
* Assign roles to users, groups, or services

---

## 🛡️ Security Concepts

* Zero Trust = never trust, always verify
* Defense in Depth = multiple layers of security

---

## 🔒 Conditional Access

* Controls access based on:

  * Location
  * Device
  * Risk level

---

## 🔐 Encryption

* Data at rest = stored data
* Data in transit = moving data

---

## 🛡️ Microsoft Defender for Cloud

* Security posture management
* Threat protection
* Recommendations

---

## ⚠️ Exam Must-Know

* Authentication ≠ Authorization
* RBAC = access control
* MFA = strongest identity protection
* Zero Trust = core security model

---

## 🎯 Final Tip

Focus on:

* Differences (Auth vs AuthZ)
* Security concepts (Zero Trust, Defense in Depth)
