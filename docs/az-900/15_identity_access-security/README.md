## 🧠 Exam Focus Areas

Pay special attention to:

- **authentication vs. authorization**
- **Microsoft Entra ID** and identity concepts
- **MFA** and passwordless authentication
- **RBAC** and **Conditional Access**
- **Zero Trust**, defense-in-depth, and encryption basics

---

## 🚀 Study Tip

For AZ-900, you should clearly distinguish between:

1. **who a user is** → authentication
2. **what a user can do** → authorization
3. **how access is protected** → security controls
