# AZ-900 Cheat Sheet – Cloud Service Types

## ☁️ Overview

Cloud services are categorized into three main models:

* Infrastructure as a Service (IaaS)
* Platform as a Service (PaaS)
* Software as a Service (SaaS)

---

## 🏗️ IaaS (Infrastructure as a Service)

### Description

Provides virtualized infrastructure such as virtual machines, storage, and networking.

### Responsibilities

* You manage:

  * OS
  * Applications
  * Data
* Azure manages:

  * Physical hardware
  * Datacenter

### Use Cases

* Full control over environment
* Custom configurations

### Example

* Azure Virtual Machines

---

## ⚙️ PaaS (Platform as a Service)

### Description

Provides a platform to develop, run, and manage applications without managing infrastructure.

### Responsibilities

* You manage:

  * Applications
  * Data
* Azure manages:

  * OS
  * Runtime
  * Infrastructure

### Use Cases

* Web apps
* APIs
* Development platforms

### Example

* Azure App Service
* Azure Functions

---

## 💻 SaaS (Software as a Service)

### Description

Provides fully managed software accessible over the internet.

### Responsibilities

* You manage:

  * Usage
  * Configuration (limited)
* Azure/provider manages:

  * Everything else

### Use Cases

* Email
* Office tools
* CRM systems

### Example

* Microsoft 365

---

## 🔄 Comparison

| Feature           | IaaS           | PaaS        | SaaS          |
| ----------------- | -------------- | ----------- | ------------- |
| Control           | High           | Medium      | Low           |
| Management effort | High           | Medium      | Low           |
| Flexibility       | High           | Medium      | Low           |
| Use case          | Infrastructure | Development | End-user apps |

---

## 🎯 Key Takeaways

* IaaS = maximum control, maximum responsibility
* PaaS = balance between control and convenience
* SaaS = minimal management, ready-to-use software

---

## ⚠️ Exam Tip

Always identify:

> Who is responsible for what?

That determines the correct answer.
