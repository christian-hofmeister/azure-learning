# Practice Questions – Cloud Service Types

## 🧠 Question 1

Which cloud model provides the most control over the operating system?

A) SaaS
B) PaaS
C) IaaS
D) Serverless

**Answer:** C

---

## 🧠 Question 2

Which model requires the least management effort?

A) IaaS
B) PaaS
C) SaaS
D) Hybrid

**Answer:** C

---

## 🧠 Question 3

Which service is an example of PaaS?

A) Virtual Machines
B) Azure App Service
C) Microsoft 365
D) Azure DNS

**Answer:** B

---

## 🧠 Question 4

In which model does Azure manage the operating system?

A) IaaS
B) PaaS
C) Both IaaS and PaaS
D) Neither

**Answer:** B

---

## 🧠 Question 5

Which model is best for running custom applications with full control?

A) SaaS
B) PaaS
C) IaaS
D) Serverless

**Answer:** C

---

## 🧠 Question 6

Which model provides ready-to-use applications?

A) IaaS
B) PaaS
C) SaaS
D) Hybrid

**Answer:** C

---

## 🧠 Question 7

Which statement is true for PaaS?

A) You manage hardware
B) You manage the OS
C) You focus on applications and data
D) You manage everything

**Answer:** C

---

## 🧠 Question 8

Which model is best for developers who want to focus on code?

A) IaaS
B) PaaS
C) SaaS
D) On-premises

**Answer:** B

---

## 🧠 Question 9

Which model has the highest flexibility?

A) SaaS
B) PaaS
C) IaaS
D) Serverless

**Answer:** C

---

## 🧠 Question 10

What is the main difference between IaaS and PaaS?

A) Cost
B) Responsibility for OS and infrastructure
C) Network speed
D) Storage size

**Answer:** B

---

## 🎯 Tip

Always ask:

* Who manages the OS?
* Who manages the infrastructure?
* How much control is required?
