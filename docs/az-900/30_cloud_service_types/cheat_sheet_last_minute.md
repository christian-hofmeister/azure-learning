# AZ-900 Cheat Sheet – Cloud Service Types (Last Minute)

## ☁️ Core Models

* IaaS → infrastructure (VMs)
* PaaS → platform (apps)
* SaaS → software (end-user)

---

## ⚙️ Responsibility

* IaaS → you manage most
* PaaS → Azure manages platform
* SaaS → Azure manages everything

---

## 🧠 Quick Mapping

* VM → IaaS
* App Service → PaaS
* Microsoft 365 → SaaS

---

## ⚠️ Exam Must-Know

* More control = more responsibility
* SaaS = least effort
* IaaS = most flexibility

---

## 🎯 Final Tip

Think:

> "Who manages what?"
