# Blob Storage – Cheat Sheet

## Structure

* Storage Account → Container → Blob

## Blob Types

* Block → default
* Append → logs
* Page → VM disks

## Security Priority

1. RBAC ✅
2. SAS ⚠️
3. Access Keys ❌

## Encryption

* Enabled by default

## Key Point

* Blob storage = **unstructured data storage**
