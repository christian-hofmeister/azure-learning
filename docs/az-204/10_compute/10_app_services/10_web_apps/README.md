# Azure App Service – Web Apps

## Overview

Azure App Service is a fully managed platform for building, deploying, and scaling web applications.

Supports:
- Web Apps
- REST APIs
- Backend services

---

## Key Features

- Platform as a Service (PaaS)
- Built-in scaling
- CI/CD integration
- Custom domains & SSL
- Authentication / Authorization support

---

## When to use

- Web applications (ASP.NET, Node.js, Python, etc.)
- REST APIs
- Backend for mobile apps

---

## When NOT to use

- Full control over infrastructure required → use VMs
- Complex container orchestration → use AKS
