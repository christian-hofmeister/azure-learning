# Cheat Sheet – App Service

## Plans

- Free / Shared → limited
- Basic → dedicated VM
- Standard → autoscaling
- Premium → high performance

---

## Scaling

- Vertical → change plan
- Horizontal → instances

---

## Deployment Options

- GitHub Actions
- Azure DevOps
- ZIP deploy
- FTP

---

## Exam Traps

- App Service = PaaS (not IaaS)
- Scaling depends on App Service Plan
- Always On requires Basic or higher
